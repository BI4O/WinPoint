# 商户数据概览升级 设计规范

> **Status:** Approved for Implementation

## 概述

将商户后台管理的"数据概览"升级为带趋势图的 Tab 式仪表盘，展示最近30天的关键业务指标走势。

## 设计决策

- **Tab 切换**：4个指标各一个 Tab，点击切换图表
- **时间维度固定**：统一显示最近30天，不提供切换
- **图表样式**：不带点的纯折线图（line chart without dots）
- **颜色方案**：暖色调

| 指标 | 颜色 | 色值 |
|------|------|------|
| 销售额 | 橙色 | #F97316 |
| WIN积分 | 金黄色 | #F59E0B |
| 订单量 | 珊瑚色 | #FB7185 |
| 库存 | 玫瑰红 | #F43F5E |

## 页面结构

```
┌─────────────────────────────────────────┐
│           数据概览（升级后）              │
├─────────────────────────────────────────┤
│  统计卡片（保持不变）                     │
│  [商品总数] [总订单] [销售额] [WIN余额]  │
├─────────────────────────────────────────┤
│  Tab: [销售额] [WIN积分] [订单量] [库存] │
├─────────────────────────────────────────┤
│                                         │
│         折线图（最近30天）              │
│         不带点，纯色折线                │
│                                         │
└─────────────────────────────────────────┘
```

## 技术实现

### 新增文件

- `components/MerchantStatsChart.tsx` - 数据概览图表组件
  - Props: `metricType: 'sales' | 'win' | 'orders' | 'stock'`
  - 内部处理 mock 数据生成（30天模拟数据）

### Mock 数据生成

为每个指标生成30天的模拟数据：
- **销售额**：随机波动，趋势上升
- **WIN积分**：波动较大
- **订单量**：与销售额相关
- **库存**：逐渐下降趋势

数据格式：
```typescript
interface StatsDataPoint {
  date: string;        // '2026-03-01'
  value: number;
}
```

### 图表配置

- 使用 Recharts `LineChart` + `Line`
- `dot={false}` 隐藏数据点
- `strokeWidth={2}` 折线粗细
- X轴显示日期（简化格式如"03-01"）
- Y轴自适应数值范围
- Tooltip 显示具体数值

## 实现任务

1. 创建 `MerchantStatsChart.tsx` 组件
2. 生成4个指标的 mock 数据（30天）
3. 实现 Tab 切换逻辑
4. 更新 `merchant/manage/page.tsx` 集成图表
