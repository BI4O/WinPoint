# WinPoint UI 改版设计文档

## 概述

对 WinPoint 演示应用进行 UI 改版，涉及导航、探索页面、兑换页面和我的页面。

## 1. 导航 Tab 改名

| 原名 | 新名 |
|------|------|
| 首页 | 主页 |
| 商家 | 探索 |
| 积分商城 | 兑换 |
| 资产 | 我的 |

**文件**: `components/Header.tsx`

## 2. 探索页面改版

**文件**: `app/merchants/page.tsx`

- 主标题：`WinPoint网购`
- 副标题：`一次过网购所有顶级品牌`

## 3. 兑换页面改版

### 3.1 价格展示

**文件**: `components/RewardProductCard.tsx`

商品卡片价格展示：
- 左边：**积分数字（大字）** + "积分"
- 右边：**+ ¥现金金额**（小字）
- 格式：`100积分 + ¥1`

### 3.2 轮播广告

**新增文件**: `components/PromotionCarousel.tsx`

- 位置：页面 Header 下方、商品列表上方
- 内容：Emoji + 彩色背景块的广告形式
- 展示3-4个热门兑换品和促销广告
- 仅展示，不可点击

### 3.3 现金券专区

**新增数据**: `mockData.ts` 中添加现金券商品

- 类型：通用现金券
- 定价：100积分 = ¥1 现金券
- 展示位置：商品列表最上方，醒目位置

### 3.4 筛选分类

**文件**: `components/CategoryFilter.tsx`

筛选选项更新：
- 全部
- 现金券 ← 新增
- 餐饮
- 运动
- 电子产品
- 母婴
- 奢侈品

## 4. 我的页面改版

**文件**: `app/dashboard/page.tsx`（重命名为 `app/my/page.tsx` 或保留原名）

### 4.1 顶部展示

- 主卡片：积分余额（大字）+ "已购买 X 件商品" subtitle
- 不显示现金/收益概念

### 4.2 订单列表

**新增文件**: `components/OrderList.tsx`

#### 订单类型区分
- 标签文字："现金购买" / "积分兑换"
- 颜色区分：
  - 现金购买：蓝色系背景
  - 积分兑换：紫色/金色系背景

#### 订单状态
| 状态 | 标签 | 颜色 |
|------|------|------|
| 待发货 | ⏳ 待发货 | 橙色 |
| 运输中 | 🚚 运输中 | 蓝色 |
| 待收货 | 📦 待收货 | 绿色 |

### 4.3 Mock 数据更新

**文件**: `lib/mock-data.ts`

- 添加 `delivered` 状态（待收货/已签收）
- 添加用户自己的订单数据（区分现金购买和积分兑换）
- 添加现金券兑换记录

## 5. 路由更新

- `dashboard/page.tsx` → `my/page.tsx`（可选，或保留原路由）
- 导航链接更新
